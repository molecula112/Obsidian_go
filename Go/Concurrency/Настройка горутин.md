Количество возможных горутин = количеству логических ядер процессора.

Узнать эту цифру можно с помощью библиотеки runtime
```go
fmt.Println(runtime.NumCPU())
```

Указать сколько горутин может выполнятся одновременно можно таким кодом
```go
runtime.GOMAXPROCS(10)
```
Тут важно учесть, что если я укажу число большее чем у меня логических ядер, все равно за пределы я не выйду.
Если указать количество 1 - то тогда горутины будут работать не параллельно а в конкурентом режиме.

Можно также в ручную переключаться между горутинами:
```go
runtime.Gosched()
```
В данном случае, вот этот код выполнит все что в нем есть без каких либо прерываний
```go
func main() {  
    fmt.Println(runtime.NumCPU())  
    runtime.GOMAXPROCS(1)  
  
    go showNumbers(10)  
  
    runtime.Gosched()  
    fmt.Println("exit")  
}  
  
func showNumbers(num int) {  
    for i := 0; i < num; i++ {  
       fmt.Println(i)  
    }  
}
```

Также планировщик может переключаться когда вызван time.Sleep, планировщик сам поймет что есть время простоя и можно переключиться. Но все эти вопросы к собеседованиям но не юзается в практике.
