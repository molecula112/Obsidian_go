#### TryLock()
Это неблокирующая попытка захватить `Mutex`
```
Lock() → ждёт пока mutex освободится  
TryLock() → пытается захватить и сразу возвращает результат
```

```go
func (m *Mutex) TryLock() bool
```
Он возвращает:
```
true - mutex успешно захвачен  
false - mutex уже занят
```
---
```go
mu.Lock()
```
Поведение:
```
если mutex занят  
goroutine блокируется  
ждёт освобождения
```

```go
if mu.TryLock() {  
	...  
}
```
Поведение:
```
если mutex занят  
сразу false  
никакого ожидания
```
