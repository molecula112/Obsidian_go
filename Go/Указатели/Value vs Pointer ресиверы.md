`Value receiver` - метод работает с копией объекта
`Pointer receiver` - метод работает с оригиналом

Когда использовать `Pointer receiver`:
1. Если метод изменяет структуру
2. Если структура большая, чтобы ее не копировать
3. Нужно единое состояние, то есть методы работают с одним объектом

Когда использовать `Value receiver`:
1. Структура маленькая
2. Метод не изменяет состояние
3. Структура immutable

Самый важный тейк:
Если метод:
- меняет данные
- или struct не маленький
→ используй pointer receiver

Важное правило:
```
Если хотя бы один метод структуры использует pointer receiver,  
лучше чтобы все методы использовали pointer receiver.
```
---
С функциями ситуация похожая:
```
func f(x T) → копия  
func f(x *T) → работа с оригиналом
```
