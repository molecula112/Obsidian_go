`Указатели это тип данных, который в качестве значения хранит адрес ячейки памяти значения  либо другого указателя`

```go
*T → тип "указатель на T"  
&x → взять адрес переменной x (получить указатель)  
*x → разыменовать указатель
```

Операторы указателей:
`&` - взять адрес
```go
x := 5  
p := &x
```

`*` - разыменовывание
```go
x := 10  
p := &x  
  
fmt.Println(*p) // 10
```

Если указатель не инициализирован — он `nil`
```go
var p *int  
fmt.Println(p) // nil
```
В таком случае разыменовывание вызовет панику, но это можно использовать так же  и в плюс, например как Optional
```go
func hasWallet(money *int) bool {  
	return money != nil  
}
```

---
Способы создать указатель:
1 ) Через `&`
```go
x := 10  
p := &x
```

2 ) Через `new()`
```go
p := new(int)  
*p = 10
```
`new(int)`:
- выделяет память
- возвращает `*int`
- значение = zero value
