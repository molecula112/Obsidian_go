Слайс - это view на массив, он не хранит элементы, а ссылается на массив:
`slice = (ptr, len, cap)`
Где:
- `ptr` - указатель на underlying array
- `len`- количество элементов
- `cap` - вместимость до перераспределения памяти
    
Пример структуры:
```go
type slice struct {  
    ptr unsafe.Pointer  
    len int  
    cap int  
}
```

Почитай и вникни, не ленись! Тут очень полезные мысли.
