Это улучшенная версия `gofmt`, он делает все то же самое что и `gofmt` + автоматически управляет импортами.
Он автоматически:
- добавляет недостающие импорты
- удаляет неиспользуемые
- сортирует импорты
- группирует стандартные / внешние / локальные пакеты

Поставить:
```shell
go install golang.org/x/tools/cmd/goimports@latest
```
