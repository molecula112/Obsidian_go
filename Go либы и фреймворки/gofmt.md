`gofmt` - это официальный инструмент форматирования кода, он автоматически приводит исходный код к стандартному стилю, принятому в экосистеме `go`.
Он автоматически:
- выравнивает отступы и пробелы
- расставляет переносы строк
- форматирует блоки кода (`if`, `for`, `func` и т.д.)
- упорядочивает и форматирует импорты
- делает код единообразным независимо от стиля автора

### Как использовать:

```shell
gofmt -w main.go
```
`-w` — перезаписывает файл

```shell
gofmt -d main.go
```
`-d` — показать diff, не изменяя файл

```shell
gofmt -w .
```
форматировать весь проект
