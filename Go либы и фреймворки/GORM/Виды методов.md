В GORM есть два типа методов:
###### Методы построения запроса
Они ничего не выполняют, только собирают SQL
```go
db.Where("id = ?", userID)  
db.Limit(10)  
db.Offset(20)
db.Order("created_at DESC")
```

###### Терминальные методы
Они выполняют SQL
```go
First()  
Find()  
Create()  
Delete()  
Updates()  
Save()  
Scan()
```
И именно здесь ORM:
1. отправляет SQL в базу  
2. получает результат  
3. записывает его по указателю
```go
var user User  
  
db.Where("id = ?", 1).First(&user)
```

#### Крутое правило
```
Find(&slice) → заполнить slice  
First(&struct) → заполнить struct  
Create(&struct) → сохранить struct и обновить поля
```
