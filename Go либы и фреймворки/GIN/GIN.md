Подтянуть можно:
```
go get -u github.com/gin-gonic/gin
```

Важная особенность фреймворка что он работает на собственной реализации роутера, использующей Radix Tree для маршрутизации запросов, что обеспечивает быстрое распределение нужного обработчика даже при большом количестве маршрутов, что критично для хайлоад микросервисов.
