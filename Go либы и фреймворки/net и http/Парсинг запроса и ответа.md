Чтобы превратить данные полученные из запроса в модель существует
```go
var user models.User  
err := json.NewDecoder(r.Body).Decode(&user)
```
Простым языком:
```
JSON из запроса → структура Go
```

И обратная модель, записать в ответ:
```go
func (h *UserHandler) CreateUser(w http.ResponseWriter, r *http.Request) {  
    var user models.User  
	...
    err = json.NewEncoder(w).Encode(user)  
    ... 
}
```
Это фактически говорит переведи в json
