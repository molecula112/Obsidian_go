Когда приходит запрос:
```
Browser → TCP connection → Go server
```

Внутри происходит примерно вот это:
```
accept TCP connection  
parse HTTP request  
create http.Request  
create ResponseWriter  
call handler.ServeHTTP()  ---> вот тут
write response
```

Как можно заметить GO буквально вызывает наш метод 
```
handler.ServeHTTP(w, r)
```
